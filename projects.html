Raspberry Pi Smart Alarm Clock

This past summer I made a tablet using a touch display and a raspberry pi computer. The motivation for this came when my advisor Peter C. Johnson introduced me to the raspberry pi and some of the many things that could be done with it, ranging from using it as a server to recreating an Atari retro gaming console.

After looking at many of the projects that were displayed on adafruit.com, I decided to start my own project using a raspberry pi. This project was a combination of a few different projects I saw, and it is still not finished. The end goal is to have a smart alarm clock, running on a touch display, that would be synced with Google Calendar, display live news headlines, show the current weather and forecast, have alarm clock capabilities, and be interactive with a voice assistant. So far, all of those are complete with the exception of the voice assistant. Moving forward, I want to sync the tablet with the lights in my dorm room so that I can control them with the voice assistant.
